# 📚 Full-Stack Labs

> Full-stack web development labs and projects — Python, Django, JavaScript, HTML/CSS

## About

A collection of lab exercises and projects from DCU's Full-Stack Development module (CA298). Covers frontend fundamentals, backend development with Django, and full-stack integration.

## Contents

| Folder | Topic |
|--------|-------|
| `Lab_1` | Introduction to web development |
| `Lab_2` | HTML & CSS fundamentals |
| `Lab_3` | JavaScript basics |
| `lab_5` | Django models & views |
| `Lab_6` | Forms & user input |
| `Lab_11` | Advanced Django features |
| `Project_1` | Full-stack pizza ordering project |

## Tech Stack

- **Backend:** Python, Django
- **Frontend:** JavaScript, HTML, CSS
- **Database:** SQLite
- **Tools:** Git, pip

## Getting Started

Each lab is self-contained. To run a Django project:

```bash
cd <lab_folder>
pip install django
python manage.py migrate
python manage.py runserver
```
